This is the bitDuino10.
  • 2013年11月　@kimio_kosaka Arduino IDE ver.1.0.5 に ATtiny10の開発環境を構築
  • 2013年11月　@kimio_kosaka ATtiny10開発環境インストールと利用手順公開
  • 2014年 4月　@maris_HY氏 @kimio_kosaka版を機能強化したbitDuinoを開発公開
  • 2016年12月　@kimio_kosaka bitDuino をArduino IDE ver.1.6.x に対応させる。
  • 2016年12月　@kimio_kosakaAttiny10とAttinu13はコンパイル環境を同一にできないためbitDuino10とBitDuino13に分離した。
  • 2017年12月　@kimio_kosaka bitDuino10, bitDuino13 を Arduino IDのボードマネージャからインストールできるようにした。
  • 2018年 1月　@kimio_kosaka bitduino10とbitDuino13のインストールパッケージを統合。（実行環境は分離）
